# 보안그룹

`위치` 인스턴스 > 네트워크 > 보안 그룹 변경

인스턴스 방화벽(인바운드/아웃바운드)을 설정한다. EC2가 시작상태에서도 가능하다.

![01](/docs/aws/ec2/securitygroup/01.png)

보안 그룹 변경 팝업에서 원하는 보안 그룹을 설정한다.
![02](/docs/aws/ec2/securitygroup/02.png)

## HTTP/HTTPS

`위치` : 네트워크 및 보안 > 보안 그룹

HTTP/HTTPS를 설정한다.
