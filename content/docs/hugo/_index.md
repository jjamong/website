# Hugo
Static Site Generator

## 정보
사이트 : [https://gohugo.io/](https://gohugo.io/)<br>
설치 사이트 : [https://github.com/gohugoio/hugo/releases](https://github.com/gohugoio/hugo/releases)


## 명령어
hugo 프로젝트 생성
```shell
hugo new site project
```

로컬서버 실행 localhost:1313
```shell
hugo server -D
```
