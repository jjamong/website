명령창에서 `hugo new site` 실행하면 다음 구성으로 디렉토리 구조가 생성된다.

```
.
├── archetypes
├── config.toml
├── content
├── data
├── layouts
├── static
└── themes
```

## 디렉토리 구조 설명

### archetypes
https://gohugo.io/content-management/archetypes/

### assets
Hugo [Pipes](https://gohugo.io/hugo-pipes/)에서 처리해야 하는 모든 파일을 저장한다.

참고 : assets 디렉토리는 기본적으로 생성되지 않음.
