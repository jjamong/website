npm은 자바스크립트를 관리하기 위한 Node.js 기본 패키지 관리자입니다.

Node.js를 사용하는 프로젝트에서 npm 명령어를 통해 공개된 모듈들을 설치/업데이트/삭제 하면서
쉽게 관리가 가능합니다.

<br>

npm이 없었을 때엔 github를 통해 수동으로 다운로드 하여 관리하곤 했었습니다.

이런 불편함을 없애고자 개발됬으며 Node.js를 설치하면 기본적으로 npm이 설치됩니다.
